# Recon Summary — OWASP Juice Shop v19.2.1

Generated: 2026-04-09T (inline recon)

## Section 1: Business Context

OWASP Juice Shop is an intentionally vulnerable web application designed for security training, demonstrations, and CTF events. It implements nearly every OWASP Top 10 vulnerability category deliberately. Real deployments exist at demo.owasp-juice.shop. The application is a full e-commerce shop with user accounts, products, orders, payment, wallet, and a chatbot. 

## Section 2: Technology Stack

- **Runtime**: Node.js v22 / TypeScript
- **Backend Framework**: Express.js 4.22.1
- **Frontend**: Angular (SPA, separate frontend/ directory, compiled to static)
- **Primary Database**: SQLite via Sequelize ORM 6.37.3
- **Secondary Database**: MarsDB (MongoDB-like in-memory, for reviews and orders)
- **Auth**: Custom JWT with express-jwt 0.1.3, jsonwebtoken 0.4.0 (RS256)
- **Session**: Cookie-parser + JWT in localStorage
- **WebSocket**: Socket.io 3.1.2
- **Security Middleware**: Helmet 4.6.0 (partial), express-rate-limit 7.5.1
- **File Upload**: Multer 1.4.5-lts.1
- **XML Processing**: libxmljs2 ~0.37.0
- **Monitoring**: Prometheus (prom-client 14.2.0), Grafana dashboard
- **Containerization**: Dockerfile (distroless Node.js 24, UID 65532), no docker-compose for prod
- **SBOM**: CycloneDX npm integration in Docker build
