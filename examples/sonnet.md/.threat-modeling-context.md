**Inferred from code and documentation:** OWASP Juice Shop is an intentionally vulnerable web application used for security awareness, training, demonstration, and exercise. It is an open-source OWASP flagship project (version 19.2.1). The application deliberately contains a large number of security vulnerabilities across all OWASP Top 10 categories and beyond, implemented as hacking challenges for users. It is **NOT** a template for secure development — it is the opposite. It is deployed publicly at https://demo.owasp-juice.shop and multiple other instances.

## Security Policy

OWASP Juice Shop is an *intentionally vulnerable* web application. Security patches are only provided for the latest released minor version (19.2.x). The project explicitly states that discovered vulnerabilities may be turned into hacking challenges rather than fixed.
