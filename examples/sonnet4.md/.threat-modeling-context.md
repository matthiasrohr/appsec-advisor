## Business Context

OWASP Juice Shop is an intentionally insecure web application for security training and CTF challenges. It is a Node.js/TypeScript/Express monolith with an Angular SPA frontend. It contains deliberately introduced vulnerabilities covering OWASP Top 10. Despite being intentional, the threats are real and this assessment models them as architectural risks.

## Architecture Notes

- Express.js server on port 3000
- Angular v14+ SPA frontend (dist served statically)
- SQLite database via Sequelize ORM
- MarsDB (in-memory MongoDB-like) for reviews/orders
- JWT authentication (RS256) using hardcoded private key in source
- No API gateway, no WAF, no BFF pattern
- Prometheus metrics exposed on /metrics (unauthenticated)
- FTP directory browsable at /ftp
- Encryption keys browsable at /encryptionkeys
