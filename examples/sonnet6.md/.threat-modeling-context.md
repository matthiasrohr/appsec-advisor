## Architecture Notes

- **Runtime**: Node.js 24 / TypeScript
- **Frontend**: Angular SPA (Angular 17+) 
- **Backend**: Express.js REST API
- **Database**: SQLite (via Sequelize ORM) + NeDB (MongoDB-like, for reviews/orders)
- **Auth**: Custom JWT (jsonwebtoken) with RSA keys
- **Deployment**: Docker (distroless base image, runs as UID 65532)
- **Key security findings**: Hardcoded RSA private key in lib/insecurity.ts, MD5 password hashing, wildcard CORS, no CSP on main app
