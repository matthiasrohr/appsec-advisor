## Business Context

docs/business-context.md not present in this repository.

OWASP Juice Shop is an open-source intentionally vulnerable web application maintained by the OWASP Foundation. It serves as:
- An awareness, training, demonstration, and exercise tool for security risks in modern web applications
- A CTF (Capture The Flag) competition platform
- A benchmark for security scanning tools

The application simulates a real e-commerce platform with: user registration/login, product catalog, shopping basket, order processing, payment processing (simulated), user feedback, file upload, admin panel, and a REST API.

## Security Policy

OWASP Juice Shop is an intentionally vulnerable web application. The SECURITY.md states:

"We do not want to be surprised by zero day vulnerabilities which are not part of our hacking challenges."

Supported versions: Only 19.2.x is supported. The application deliberately contains vulnerabilities but does not want unknown zero-days introduced by dependencies.
